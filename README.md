# Rust Chess Engine

A chess engine written in Rust as a learning project.

## Features

- Board representation using a 64-square array
- Move execution
- Attack detection
- Check detection
- Legal move generation
- Checkmate and stalemate detection
- Pawn promotion
- Comprehensive unit test coverage

## Planned Features

- Castling
- En passant
- Perft
- 50 move rule and 3 move repetition
- Position evaluation
- Minimax with alpha-beta pruning
- UCI support
- GUI

cargo run
