# Rust Chess Engine

A chess engine written in Rust as a learning project.

## Features

- Board representation using a 64-square array
- Pseudo-legal move generation
- Move execution
- Attack detection
- Check detection
- Unit tests for move generation and board logic

## Planned Features

- Legal move generation
- Checkmate and stalemate detection
- Castling
- En passant
- Pawn promotion
- Minimax with alpha-beta pruning
- Position evaluation
- UCI support
- GUI

cargo run
